Welcome to rNGCircos

A R package from NG-Circos

NG-Circos is developed in JavaScript and displayed in html, based on D3 and jQuery libraries.

Since R language is much more friendly in data visualization and plot programming than html, we developed this R package based on NG-Circos and htmlwidgets framework.

  • Click to install

Features

Interactive Circos Plot

rNGCircos can display multi-track and interactive Circos plot in a coding-friendly manner.

No Perl or JavaScript experience is required !

10+ Plot type

rNGCircos support multiple common types of plot including bubble, chord, heatmap, histogram, line, lollipop, point...
rNGCircos also customize several plots for popular bioinformatic data format including wig, cnv, snp, gene...

10+ Animation and Mouse Event

rNGCircos apply opening animation for every plot function. Animation of every plot is highly customizable.
rNGCircos also support customizing mouse event and mouse tooltips for plot function.

Compatible with other package

rNGCircos use data frame as input, which is compatible with output from other packages
