Quick Start

1. Installation

rNGCircos is available at github or you can download here

Install from Github

# You need devtools for that
if (!require('devtools')){install.packages('devtools')}

devtools::install_github('mrcuizhe/rNGCircos')

# R package of htmlwidgets, RColorBrewer, plyr, jsonlite
# and grDevices are required !

Install from local package

#Download rNGCircos package to local disk first!

install.packages("PATH/TO/PACKAGE/rNGCircos_X.X.X.tar.gz", repos = NULL, 
type = "source")

# R package of htmlwidgets, RColorBrewer, plyr, jsonlite
# and grDevices are required !

2. Draw an interactive Circos plot using rNGCircos

If you are a NG-Circos user and familiar with the modules and its data structure, it will only take you a few moment for getting used to rNGCircos.

Same as NG-Circos, a main function called rNGCircos() is necessary to display the genome for Circos plot, which is the coordinates basis for all modules.

Up to 20 types of modules can be added to genome track through a parameter in rNGCircos() called moduleList. Modules can be added by simply using "+" in moduleList. For example, moduleList=rNGCircosText()+rNGCircosBackground() means apply one TEXT module and one BACKGROUND module together to genome track. The other parameters in rNGCircos() are all correspond to that in NG-Circos one to one.

Note that 3 of all modules(COMBINATION, COMPARE, REDIRECT) , their parameters are included in rNGCircos()
For the other 17 of 20 modules, which has an independent function(rNGCircosArc(), rNGCircosAuxLine(), rNGCircosBackground(), rNGCircosBubble(), rNGCircosChord(), rNGCircosCnv(), rNGCircosGene(), rNGCircosHeatmap(), rNGCircosHistogram(), rNGCircosLegend(), rNGCircosLine(), rNGCircosLink(), rNGCircosLollipop(), rNGCircosScatter(), rNGCircosSnp(), rNGCircosText(), rNGCircosWig()), all data(if yes) are input through a parameter called data in a data.frame structure. All other parameters are all correspond to that in JacaScript version one to one.

3. Genome configuration

Default genome track

library(rNGCircos)

rNGCircos(zoom=FALSE)

Plot viewer

  • Interactive demo

Try changing genome list and its color

The genome is set by default as hg19, the genome can be changed by setting genome=list("Example1"=100,"Example2"=200,"Example3"=300). Meanwhile, the color of each chromosome can be changed by setting genomeFillColor = c("red","blue","green")

library(rNGCircos)

rNGCircos(zoom = FALSE,genome=list("Example1"=100,"Example2"=200,
"Example3"=300),genomeFillColor = c("red","blue","green"))

Try automatically filling genome color

The filling color for chromosome can also be set automatically by choosing a RColorBrewer palette, such as Blues

library(rNGCircos)

rNGCircos(zoom = FALSE,genome=list("Example1"=100,"Example2"=200,
"Example3"=300),genomeFillColor = "Blues")

5. Plot saving

If you are using RStudio, the plot can be saved as an image or a html page directly from viewer panel.

If you are using R in command, the plot can be saved as a html page, thanks to the htmlwidgets library

library(rNGCircos)
library(htmlwidgets)

plot <- rNGCircos(zoom=FALSE,genome=list("Example1"=100,"Example2"=200,
"Example3"=300),genomeFillColor = "Blues",chrPad = 0,genomeTicksDisplay = TRUE,
genomeTicksScale = 10,genomeLabelDisplay = FALSE)

saveWidget(plot, "plot.html")

The plot will be saved at your current directory
