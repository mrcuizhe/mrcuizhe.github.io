Welcome to interacCircos

Designed for visualization of interactive Circos plot

Inspired by circosJS, BioCircos.js and NGCircos and developed based on htmlwidgets framework.

  • Click to install

Features

Interactive Circos Plot

interacCircos can display multi-track and interactive Circos plot in a coding-friendly manner.

No Perl or JavaScript experience is required !
Same structure and logic as ggplot2.

10+ Plot type

interacCircos support multiple common types of plot including bubble, chord, heatmap, histogram, line, lollipop, point...
interacCircos also design several plots for popular biological data format including wig, cnv, snp, gene...

10+ Animation and Mouse Event

interacCircos apply opening animation for every plot function. Animation of every plot is highly customizable.
interacCircos also support customizing mouse event and mouse tooltips for plot function.

Compatible with other package

interacCircos use data frame as input, which is compatible with output from other packages
